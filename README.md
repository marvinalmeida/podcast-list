# PODCAST LIST

Projeto inicializado com o **Create React App** _CRA_.

**Principais libs utilizadas**: styled-components, axios, react-router-dom, date-fns.

### Step 1:

```
npm install
or
yarn
```

### Step 2:

```
npm start

or

yarn start
```

O projeto ficará disponível no endereço: http://localhost:3000
